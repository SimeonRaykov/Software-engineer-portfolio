Excel-Charts (El-Ex systems)

Huge internal system. The system is used for data importing (clients, profiles, readings, hour-readings, graphs, etc.) from different excel type formats and it can visualize it using different menus/calendars/charts/tables. After the required readings/graphs have been imported the system can now calculate imbalances for individual or multiple clients depending on the user preferences. The user can choose to export readings/graphs/imbalances for any client(s) and filter them by any criteria and download the file as an excel-table. The system shows a preview of how the information is going to look like in excel and the user can choose to modify the table before downloading it.
